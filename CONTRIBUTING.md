# Contributing to Career Reality Checker

Thank you for your interest in contributing to Career Reality Checker!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/career-reality-checker.git`
3. Install dependencies: `npm install`
4. Create a new branch: `git checkout -b feature/your-feature-name`

## Development Guidelines

### Code Style

- Use TypeScript for all code
- Follow the existing code style and formatting
- Run `npm run lint` before committing

### Type Checking

- Ensure all code passes type checking: `npm run type-check`
- Avoid using `any` types unless absolutely necessary

### Testing

- Write tests for new features
- Ensure all tests pass before submitting a PR

### Commit Messages

- Use clear, descriptive commit messages
- Reference issue numbers when applicable

## Pull Request Process

1. Ensure your code follows the project's style guidelines
2. Update documentation as needed
3. Add tests for new functionality
4. Ensure all checks pass
5. Submit your pull request with a clear description

## Questions?

Feel free to open an issue for questions or discussions about the project.
