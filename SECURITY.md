# Security Policy

## Supported Versions

Only the latest version of Career Reality Checker is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

**Email**: [Your email address]

Please do not open public GitHub issues for security vulnerabilities.

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Time

We aim to respond to security reports within 48 hours and provide an update on the status of the vulnerability.

## Security Considerations

Career Reality Checker:
- Does not store user data
- Does not require authentication
- Processes data client-side or in-memory
- Does not send data to external services
- Does not use cookies or tracking

If you discover a vulnerability that could affect users, please report it immediately.
